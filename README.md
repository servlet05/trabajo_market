## 📖 Descripción

MaestroMX es una plataforma web tipo "Craigslist" enfocada en trabajos de oficios (electricistas, plomeros, técnicos, etc.). Diseñada para ser:

- **Simple**: Interfaz limpia con colores naranja y blanco
- **Segura**: Trabajos verificados manualmente por el administrador
- **Gratuita**: Sin comisiones ni costos ocultos
- **Replicable**: Puedes usarla en tu ciudad o país

## ✨ Características

- 📋 **Lista de trabajos** con categorías por oficio
- 🔍 **Búsqueda y filtros** por categoría y ubicación
- 📞 **Contacto directo** con el cliente (teléfono, WhatsApp, email)
- 🔐 **Panel administrativo** para gestionar trabajos
- 🎨 **Diseño responsive** (funciona en celulares)
- 🌍 **Multi-ciudad/país** (configurable)
- 🛡️ **Trabajos verificados** con código único
- 📊 **Estadísticas en tiempo real** de trabajos disponibles

## 🚀 Instalación Rápida

### Opción 1: GitHub Pages (Recomendado para empezar)

1. **Fork este repositorio**
2. **Ve a Settings > Pages**
3. **Selecciona rama `main` y carpeta `/ (root)`**
4. **¡Listo!** Tu sitio estará en `https://tusuario.github.io/maestromx`

### Opción 2: Servidor Local

```bash
# Clona el repositorio
git clone https://github.com/tuusuario/maestromx.git
cd maestromx

# Usa cualquier servidor web (Python, Node, etc.)
# Ejemplo con Python:
python -m http.server 8000

# Abre en tu navegador: http://localhost:8000
